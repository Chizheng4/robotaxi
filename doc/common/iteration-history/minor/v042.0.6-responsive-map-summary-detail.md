# v042.0.6 响应式地图摘要与详情

## 目标

让桌面与手机都能查看地图对象摘要和详情，同时遵循鼠标与触摸设备各自自然的交互方式，并让中控台默认优先展示完整地图。

## 完成内容

1. 中控台首次进入、重新进入和运行态恢复时，详情统一默认收起。
2. 桌面悬浮显示摘要，点击对象展开详情，点击地图空白收起详情。
3. 手机第一次轻触显示带安全边距的自适应摘要，“查看详情”或再次轻触打开详情。
4. 摘要复用 `mapSceneService`，只增加触摸展示状态，不保存业务数据。
5. 地图拖动关闭摘要；详情打开后不残留浮动层。
6. Robotaxi 继续进入专用对象检查器，不改变业务对象或模拟运行。

## 验证

- 1440 × 900 桌面真实浏览器交互链通过。
- 390 × 844 手机真实浏览器交互链通过，摘要未超出可视区域。
- 默认收起、对象展开、空白收起和再次打开均通过浏览器合同。
- 本轮未新增字段、状态或枚举，无需修改统一字段字典。
