# v038.1 Fleet Operations 触发链路与策略结果小迭代

## 1. 目标

本轮修复 v038.0 中运维策略和任务触发链路的用户理解问题：

1. Robotaxi 管理页增加人工运维触发入口，便于按单车即时验证任务单生成。
2. 运维策略恢复为三层结构：策略配置、策略执行、策略结果。
3. 策略执行的一对多结果由策略结果承接，一条结果对应一台 Robotaxi 和 0/1 个任务单。
4. 任务单补齐触发来源字段，区分策略触发和 Robotaxi 直接触发。
5. 清理策略页重复执行入口，保留行级执行和参数配置能力。
