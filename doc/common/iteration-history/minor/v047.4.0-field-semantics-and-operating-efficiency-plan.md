# v047.4.0 字段语义与经营效率指标迭代计划

## 目标

建立经营目标、预测结果和经营分析共用的字段语义与指标血缘合同，消除同一经营含义多字段、多中文名和页面硬编码，并在缺少事实时保持数据诚实。

## 任务

- [x] 核查经营目标、长期预测、经营指标和统一经营数据池的字段及事实来源。
- [x] 建立轻量字段语义注册服务，声明所属对象、定义、单位、枚举、指标概念和兼容关系。
- [x] 将目标任务利用率迁移为目标订单服务时间利用率，并更新预测计算公式和历史快照兼容。
- [x] 统一订单履约率、订单服务时间利用率及规划模式中文语义。
- [x] 在统一经营数据池增加订单服务时间利用率目标与实际比较，缺少连续状态事实时显示数据不足。
- [x] 同步字段规则、字段字典、经营规划和经营指标设计文档。
- [x] 增加字段语义、枚举隔离、迁移、公式血缘和数据不足合同验证。
- [x] 完成 bundle、提交前检查及桌面与手机真实页面验证。
- [x] 更新版本记录、提交并创建版本标签。

## 验收

1. 关键经营字段只在语义注册服务定义一次，页面通过统一展示服务获取中文和枚举。
2. 新经营目标和预测结果不再写入含义模糊的旧目标任务利用率字段。
3. 预测公式明确使用订单服务时间利用率，旧快照可以无损迁移。
4. 经营分析使用同一指标概念连接目标和实际，不复制底层事实。
5. 缺少 Robotaxi 连续状态区间时不生成虚假实际利用率。
6. 桌面和手机端经营目标、预测结果及经营分析页面均可正常加载和展示。
