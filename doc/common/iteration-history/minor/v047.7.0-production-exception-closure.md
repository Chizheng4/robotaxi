# v047.7.0 生产异常与补产闭环

## 价值目标

把生产供应从“计划数量全部生产、全部质检通过”的理想演示，升级为可以表达实际完成量、部分合格、质量失败和补产的业务闭环，同时保持供应决策、成本事实、Robotaxi 资产和供应跟踪口径一致。

## 完成内容

- [x] 需求预测冻结预测周期内可生产量与可交付量，供应决策消费该快照并形成预计供应成本。
- [x] 生产计划冻结单车标准生产成本和计划成本，生产批次按实际完成量形成实际生产成本。
- [x] 生产完成支持实际数量，质量检验支持全部合格、部分合格和全部失败。
- [x] 合格数量形成 Robotaxi 资产；失败数量形成质量损失与合格单车成本。
- [x] 少产和质检损失补入未下达排期，超产冲减后续排期，失败批次事实保留。
- [x] 生产计划以累计质检合格资产达到计划数量且无未完成批次作为完成条件。
- [x] 生产计划和生产批次接入状态时间线、成本、排期和资产详情。

- [x] 字段、枚举、业务文档、代码字典和中文展示同步完成。

## 数量闭环

```text
计划数量 -> 分期排程 -> 批次计划数量 -> 实际完成数量
        -> 合格数量 + 失败数量
        -> Robotaxi 资产 + 质量损失
        -> 不足补产 / 超产冲减后续排期
```

## 架构边界

- 生产计划保存规划和成本快照，生产批次保存实际发生事实，二者不相互覆盖。
- 质检失败按质量损失处理，补产形成新排程和新批次，不删除失败批次。
- 实际生产成本只通过统一成本记录进入经营事实，供应决策预计成本和计划成本不冒充实际成本。
- 本版本不进入生产工序、物料、人工工位和返工作业内部，也不接入模拟运行主扫描。
