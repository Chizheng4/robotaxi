# v047.1.3 统一图表滚动与响应式迭代计划

## 目标

从共享数据图表控件根治长序列接管页面纵向滚动的问题，使预测结果、经营分析以及后续新增图表在桌面和手机端使用同一交互合同。

## 任务

- [x] 删除长序列图表对页面纵向滚轮的接管，日、周、月使用相同交互合同。
- [x] 预测结果和经营分析继续统一调用共享图表控件，不增加页面私有修复。
- [x] 手机端保持纵向触摸滚动、点击数据提示和无页面级横向溢出。
- [x] 扩展统一图表合同，覆盖短序列、长序列、滚轮和触摸行为。
- [x] 重建 bundle，完成桌面端、手机端真实页面与提交前检查。

## 完成标准

长序列和短序列图表均不接管页面纵向滚动；桌面鼠标提示、手机点击提示和响应式宽度保持正常；预测结果与经营分析不存在页面私有图表实现。
