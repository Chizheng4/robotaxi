# v043.0.1 统一数据趋势图迭代计划

状态：已完成

## 目标

建立预测分析与经营分析共同使用的数据图表控件，使时间刻度、数值单位、悬浮提示、触摸交互、长周期浏览和响应式行为成为默认交付能力，同时保持现有计算服务和模拟运行不变。

## 任务

- [x] 建立图表数据模型、坐标、刻度和性能适配服务；
- [x] 实现统一折线 / 柱状图控件及中文浮动数据；
- [x] 接入需求预测的四张趋势图；
- [x] 接入经营分析的订单趋势和时段分布；
- [x] 更新前端趋势图默认交付规则；
- [x] 验证桌面鼠标、手机触摸、内部横向滚动、无页面溢出和现有业务回归；
- [x] 更新 README、版本、站内更新记录并归档提交。

## 验收

- 日、周、月或分类标签可读，长序列只在图表内部移动；
- 鼠标、触摸和键盘均可读取时间、指标、数值和单位；
- 预测与经营分析共用同一控件，不保留两套图表渲染逻辑；
- 切换粒度不重新执行业务计算，手机与桌面无整体横向溢出；
- 完整提交前检查和真实浏览器合同通过。

## 验证结果

- 一万点图表模型构建约 `0.15ms`，展示节点限制为 360 个；
- 桌面四图、完整周期刻度和鼠标数据浮层通过；
- `390 × 844` 手机视口图内横向浏览、触摸浮层和页面零横向溢出通过；
- 预测计算、经营分析数据源与模拟运行逻辑未修改。
