# v047.7.1 供应能力口径与图表提示

## 目标

- 区分需求缺口、预测期供应能力和最终生产计划数量。
- 修复预测结果与草稿生产计划继续继承旧错误能力快照的问题。
- 统一数据图表为单一提示层，避免跨图表累积多个浮动提示。

## 任务

- [x] 生产画像独立计算预测期可生产、可交付和可供应能力上限。
- [x] 需求预测、供应决策和生产计划复用同一供应能力函数。
- [x] 草稿计划更新最新能力口径，已确认计划保留历史快照。
- [x] 预测结果调整供应数量展示顺序并迁移历史错误口径。
- [x] 删除重复图表提示实现，增加统一提示所有权与释放生命周期。
- [x] 同步字段语义、生产画像和生产计划方案文档。
- [x] 完成计算合同、历史兼容、图表性能与真实页面验证。

## 验收

- 预测期可生产和可供应数量不被建议生产数量截断。
- 供应决策计划数量不超过需补充供应、生产能力和交付能力。
- 草稿生产计划展示统一重算值，确认后快照不被回改。
- 任意时刻最多显示一个图表提示，离开或切换图表后旧提示被释放。
- 桌面与手机页面正常加载，预测结果和生产计划核心字段可读。
