# v042.0.10 项目说明与供应展示一致性

## 目标

让项目说明成为网站内可阅读内容，并修复生产计划选择态和供应模块中文展示的一致性问题。

## 完成内容

- 当前项目 `README.md` 作为正式项目文档纳入版本，并保持为网站项目说明的唯一内容源。
- 登录用户菜单增加“项目 README”，采用右侧轻量浮层展示，支持内部滚动、关闭和手机窄屏适配。
- 新增轻量 README 读取与解析服务，不引入大型 Markdown 依赖；生产发布包明确包含 README。
- 页面从菜单或工作台重新进入时清除历史行选择并收起详情，浅蓝色只表达用户本次真实选择。
- 补齐经营规划和供应管理对象类型的中文值字典，详情与最近事件不再透传内部英文对象名。
- 字段字典文档、代码字典和字段展示合同同步更新。

## 验证

- `bash scripts/check-before-commit.sh` 通过。
- GitHub Pages 构建与 README 发布产物合同通过。
- 桌面和 `390 × 844` 手机真实浏览器验证项目说明浮层可读、可滚动、可关闭且不越界。
- 现有地图、详情、列表选择和操作菜单交互回归通过。
