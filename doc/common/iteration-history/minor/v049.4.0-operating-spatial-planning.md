# v049.4.0 运营区域规划闭环

## 目标

在不接管网格仿真业务运行的前提下，把城市地理中的运营区域规划从可见入口升级为可实际使用、可校验、可发布和可刷新恢复的桌面端空间编辑闭环。

## 完成内容

1. 城市地理与网格仿真共用场景切换和地图导航工具合同。
2. 手机端只查看已发布空间对象，不显示规划入口和几何编辑器。
3. 已发布 Zone、地点和服务区域可载入当前边界并发布新版本，旧版本保留已替代关系。
4. 城市对象使用独立稳定编号和统一目录版本，发布前校验完整候选目录。
5. 编辑已有对象时自动聚焦边界，并兼容 Terra Draw 的特征返回结构。

## 真实操作验收

在桌面浏览器中选择“商业中心”地点，依次执行调整边界、拖动顶点、形成草稿、校验和发布；发布后刷新页面，再次选择该地点可以载入最新有效边界。实测过程中发现并修复了绘图特征编号读取错误和编辑对象未自动聚焦问题。

## 边界

- 城市空间规划只更新城市空间目录和版本化方案。
- 网格 Zone、Place、ServiceArea、订单、任务和模拟运行不因本次发布改变。
- 手机端不提供复杂空间规划，后续继续保持轻量浏览和详情体验。
