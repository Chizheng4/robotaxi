# v042.0.2 手机地图标注与信息密度

## 目标

让手机地图保留与桌面一致的对象标注语言，同时避免窄屏标签堆叠、首屏无有效信息和操作体验下降。

## 完成内容

1. 手机地图默认视口聚焦当前运营区。
2. 所有地图标签收敛到独立标注图层。
3. 手机首屏只显示 Zone 与地点锚点标签。
4. 道路与服务区域标签在手机首屏隐藏，完整信息由点击详情承接。
5. 细节缩放后隐藏区域与地点标签，避免放大时文字干扰。
6. 保留车辆、Cell 选择、拖动、缩放、复位和详情联动。

## 验证

- 手机真实浏览器可视区域至少显示一个空间标签。
- 20 个存在当前位置的 Robotaxi 点位全部生成。
- Cell 点击后存在唯一选中框且悬浮提示关闭。
- 手机页面无全局横向溢出。
- 完整提交前检查通过。
