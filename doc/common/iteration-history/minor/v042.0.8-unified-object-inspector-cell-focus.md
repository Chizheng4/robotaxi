# v042.0.8 统一对象详情与 Cell 聚焦

## 目标

将右侧详情收敛为可组合的统一对象检查器，并让地图业务对象选择与空间 Cell 聚焦同时成立。

## 完成内容

- 详情统一为标题、可选摘要、可选页签、内容视口、简单字段、复杂字段和专用内容组合，所有内容共享统一缩进、字号、行距和滚动边界。
- 保留 Robotaxi 专用运营摘要和分类页签，但接入同一详情控件，不建立独立页面结构。
- 字段字典缺失时不再把内部英文 key 直接显示到页面，并输出开发提示供字段合同定位。
- 地图对象选择与 Cell 聚焦分离：点击地点、服务区域、道路、运营中心或 Robotaxi 时，右侧显示对象详情，地图同时标出实际点击 Cell。
- 浏览器合同新增详情基线、对象 Cell 选中、桌面鼠标和手机触摸验证。

## 验证

- `bash scripts/check-before-commit.sh` 通过。
- `1280 × 720` 桌面真实浏览器地图交互通过。
- `390 × 844` 手机真实触摸交互通过。
- 地图 10,000 辆 Robotaxi 投影和双区域场景性能保持原水平。
