# v043.0.7 线上演示数据自动引导计划

状态：已完成，待发布

## 目标

确保线上访客使用最新发布资源，并在首次无数据时自动形成可展示的需求预测与模拟经营结果。

## 任务

- [x] 建立部署版本轻量检查与单次刷新；
- [x] 建立线上环境和数据缺失判断服务；
- [x] 复用经营规划服务补齐需求预测结果；
- [x] 复用模拟运行服务创建并启动缺失运行；
- [x] 保持已有数据不覆盖、不重复执行；
- [x] 完成服务合同与真实生产预览浏览器验证。

## 验收

- 全新线上访客登录后可以自动获得预测结果和模拟运行；
- 已有预测结果或已执行模拟时不重复执行；
- 本地开发默认不触发演示初始化；
- 模拟运行性能、业务闭环和经营数据自动计算链路保持不变。
