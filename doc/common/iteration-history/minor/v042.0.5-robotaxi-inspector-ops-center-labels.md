# v042.0.5 Robotaxi 检查器、运营中心与地图标签

## 目标

恢复 v041.3 已确认的 Robotaxi 专用对象检查器，扩大 Zone 1 运营中心真实空间范围，并解决服务区域名称遮挡地点和运营中心名称的问题。

## 完成内容

1. Robotaxi 地图选择继续进入统一详情链路，并显示身份、电量、位置、当前任务和下一任务专用摘要。
2. Robotaxi 详情保留基础信息、资产事实、任务状态、位置上下文和行驶记录五类页签。
3. `P-006 / OC-001` 从 2 × 2 扩为 3 × 4、共 12 个同源 Cell。
4. 运营支持子区同步扩容，五类运维职能区域重新分配到运营中心内部 Cell。
5. 服务区域名称只在细节缩放出现；出现时隐藏地点名称，形成互斥语义层级。

- 浏览器确认 Robotaxi 专用摘要和五类详情页签均存在。
