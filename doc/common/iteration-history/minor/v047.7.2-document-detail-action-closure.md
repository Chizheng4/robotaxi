# v047.7.2 供应单据交互闭环

## 目标

修复生产计划、生产批次和交付单中失效的详情入口，并统一表格复杂单元格与行选中的交互边界，补齐生产至交付操作后的用户反馈和下游单据定位。

## 完成范围

- 三类供应单据的查看详情统一接入页面、对象和主键上下文。
- 操作列统一使用标准行操作控件与紧凑宽度。
- Robotaxi 编号列表、排队任务等可交互单元格阻止事件冒泡，不再同时打开右侧详情。
- 生产计划确认、生产批次生成、生产、质检和交付动作补充反馈；下游单据生成后直接进入对应页面。
- 生产计划生成生产批次时写入可追溯的最近任务事件。

## 验证

- 行操作与计划确认专项验证通过。
- 生产供应成本闭环专项验证通过。
- 生产异常与补产专项验证通过。
- `src/main.bundle.js` 已重新构建并通过语法检查。
- 1280 x 720 与 390 x 844 真实浏览器加载验证通过。

本版本未修改生产、质检、交付业务状态机，也未改变模拟运行主路径。
