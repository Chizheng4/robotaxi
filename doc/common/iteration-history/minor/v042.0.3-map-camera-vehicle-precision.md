# v042.0.3 地图相机与车辆视觉精度

## 目标

解决地图复位后 Cell 过大、车辆退化为白色圆点、详情展开遮挡地图以及旧快照详情名称过期的问题，同时保持空间对象、Robotaxi 与模拟运行边界不变。

## 完成内容

1. 建立基于地图边界和工作区宽高比的复位相机，桌面完整观察双区域，手机继续聚焦当前运营区。
2. 标签尺寸同时适配相机范围和地图缩放，详情展开或收起后保持稳定信息密度。
3. Robotaxi 恢复轻量小车图片标记，仍由统一车辆投影服务提供当前位置。
4. 道路概览层级连续显示，节点只在细节层级出现；Cell 选中反馈增强。
5. Zone 边界只响应边线交互，不再阻挡内部 Cell；选择对象后关闭悬浮摘要。
6. 地图与详情统一合并当前空间目录，旧运行快照不再覆盖新版空间名称。
7. 增加规划态跨区域外围环路，以标准道路、片段、节点和 Cell 连接两区，并保留中部主连接通道。

## 验证

- 完整提交前检查通过。
- 1440 × 900 真实浏览器地图合同通过，20 个车辆图标全部生成。
- 手机视口地图合同通过，无页面级横向溢出。
- 地图静态节点预算、Cell 唯一选中和悬浮信息关闭合同保持通过。
