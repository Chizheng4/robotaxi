# v047.6.0 生产供应与成本闭环

## 目标

把供应决策形成的生产计划落实为可分期下达、可追溯生产与质检、可形成 Robotaxi 资产和实际成本事实的业务闭环；不接入模拟运行主扫描。

## 固定边界

1. 生产工厂是独立资源对象；当前初始化一个工厂、一条生产线。
2. 生产画像描述工厂在一个版本周期内的能力，不是生产事实。
3. 生产计划确认时冻结工厂、生产画像、单位生产成本和分期排程快照。
4. 生产批次只承接一个排程期次；当前生产阶段完成且下一期已到下达日期后，才能生成下一批次。质量检验可与下一期生产并行。
5. 生产完成生成一条生产成本事实；质检通过只更新合格数量、质量损失和合格单车成本，不重复生成总成本。
6. 质检通过后形成 Robotaxi 资产，并继承生产批次的合格单车成本；交付和运营准入继续使用各自独立单据服务。
7. 供应成本分析只读取统一成本记录、生产批次和 Robotaxi 事实，不创建第二份成本事实。

## 生命周期

- 生产计划：`草稿 → 已确认 → 执行中 → 已完成`，草稿可取消。
- 生产批次：`待生产 → 生产中 → 待质检 → 质检中 → 已完成 / 质检失败`。
- 下一批次下达条件：计划已确认或执行中、无待生产或生产中的当前批次、下一排程期次到期、仍有未下达数量。

## 成本口径

- 标准生产成本：230,000 元/辆，作为当前模拟规划基准，可在成本配置中调整。
- 计划预算成本 = 计划数量 × 计划冻结单位成本。
- 批次实际生产成本 = 生产完成数量 × 批次冻结单位成本。
- 合格单车生产成本 = 批次实际生产成本 ÷ 质检合格数量。
- 质量损失成本 = 批次实际生产成本 - 合格数量 × 批次冻结单位成本。
