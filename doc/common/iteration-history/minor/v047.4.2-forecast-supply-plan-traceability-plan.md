# v047.4.2 预测与生产计划追溯迭代计划

## 目标

固定预测结果、供应决策执行和生产计划的一对一来源关系，建立同区域重叠周期的计划互斥、取消边界和统一详情。

## 任务

- [x] 供应决策执行与生产计划显式冻结预测结果、经营目标、Zone 和预测周期字段。
- [x] 新预测结果可独立执行供应决策，当前预测已有计划时只进入查看。
- [x] 同区域重叠周期的新决策自动取消旧草稿，已确认计划阻止新计划生成。
- [x] 已形成生产批次的生产计划禁止取消，未执行计划支持人工取消。
- [x] 供应决策执行和生产计划接入统一详情、字段解释、快照与计算过程。
- [x] 增加旧数据迁移、服务合同、页面动作、桌面和手机端验证。

## 验收

- [x] 每条供应决策执行唯一引用一条预测结果和一条输出生产计划。
- [x] 每条生产计划可追溯预测结果、供应决策执行、策略、画像和预测周期。
- [x] 同一经营目标、Zone 和重叠周期最多一条已确认计划。
- [x] 生产执行事实不会被取消或新预测静默覆盖。
- [x] 预测结果动作和提示与当前计划关系一致。
- [x] 提交前检查、模拟运行回归及桌面和手机页面验证通过。
