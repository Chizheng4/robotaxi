# v047.4.4 行操作与生产计划确认时点

## 目标

- 统一单功能和多功能行操作的主按钮位置、样式与直接操作方式。
- 将重叠生产计划的自动取消从创建阶段后移到计划确认阶段。

## 任务

- [x] 公共行操作控件改为主动作加更多菜单。

- [x] 重建前端并完成合同、桌面、手机和完整提交前验证。

## 验收

- 单功能与多功能的主动作使用同一按钮样式和左侧基线。
- 多功能记录可直接执行主动作，并能通过更多按钮发现次要动作。
- 两个重叠草稿可同时存在，确认其中一个后另一个自动取消。
- 已确认冲突、生产批次防护和来源追溯不受影响。
