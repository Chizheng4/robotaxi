# v043.0.2 经营分析滚动体验迭代计划

状态：已完成

## 目标

让经营分析页面在固定工作台内完整纵向浏览，并为末尾内容保留稳定阅读空间，同时不改变指标计算、业务服务和模拟运行。

- [x] 建立经营分析共享滚动画布类型；
- [x] 统一末尾阅读留白和指标明细展开高度；
- [x] 更新前端分析画布规则；
- [x] 重建 bundle 并验证桌面、低高度窗口和手机视口；
- [x] 更新版本记录、站内更新记录并归档提交。

## 验收

- 内容超过主区域高度时只在分析画布内纵向滚动；
- 滚动到末尾后保留至少 32px 阅读留白；
- 页面没有横向溢出，既有列表、地图和业务逻辑不受影响；
- 完整提交前检查和真实浏览器验证通过。

## 验证结果

- `1280 x 480` 低高度视口可滚动至末尾，末尾留白约 32px；
- 正常桌面和 `390 x 844` 手机视口无横向溢出；
- 完整提交前检查通过，经营指标计算与模拟运行代码未修改。
