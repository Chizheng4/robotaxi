# v047.1.0 预测与供应决策执行合同计划

## 目标

统一长期预测的画像参数、Robotaxi 资产口径、执行幂等、供应计划字段和分析交互，使“画像与目标 -> 预测结果 -> 供应决策 -> 生产计划”可解释、可追溯且不可重复误操作。

## 任务

- [x] 更新需求画像初始化值，并保持已有用户配置不被静默覆盖。
- [x] 按运营中心与 Zone 关系统计非退役可规划资产，保留即时可运营数量作为独立解释字段。
- [x] 统一生产计划字段与预测、供应决策执行溯源。
- [x] 为需求预测和供应决策建立输入版本指纹与幂等动作状态。
- [x] 供应决策只从明确预测结果触发，成功后进入生产计划，失败显示中文原因。
- [x] 共享图表不再接管普通纵向滚动，桌面与手机保持数据提示和页面滚动。
- [x] 预测结果改用可扩展选择器，计算过程使用紧凑标准折叠入口。
- [x] 补充领域、服务、字段、前端和真实页面合同验证。

## 验收

1. 重置后画像初始化为 0.6 / 0.7 / 0.4，保存配置仍可正确重算。
2. Zone 资产数量来自所属运营中心非退役 Robotaxi，不因待准入或短期运维状态错误归零。
3. 同一输入重复执行不新增预测结果、供应决策失败记录或生产计划。
4. 生产计划可追溯预测结果与供应决策执行，关键数量字段均有值。
5. 图表区域内普通滚轮可滚动页面，鼠标和触摸数据提示仍可使用。
6. 桌面与手机可选择多条预测结果、展开计算过程并完成供应决策。
