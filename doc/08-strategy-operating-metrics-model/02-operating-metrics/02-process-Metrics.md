# 经营过程与效率指标设计

过程指标解释经营结果由什么执行过程产生。每个分子和分母必须属于同一种业务对象与统计周期。

## 第一阶段目录

|唯一中文名|计算口径|质量规则|
|---|---|---|
|Robotaxi 分配成功率|成功选择 Robotaxi 的服务订单数 / 进入匹配的服务订单数|没有匹配订单时置空|
|路径规划成功率|成功生成路径的路径规划执行数 / 路径规划执行总数|按执行编号去重|
|履约完成率|已完成履约行驶记录数 / 已创建履约行驶记录数|终态失败和取消进入分母|
|订单履约率|已完成服务订单数 / 已创建服务订单数|与履约完成率保持不同对象语义|
|平均履约距离|完成订单有效履约距离合计 / 具有有效距离的完成订单数|缺失距离不按 0，单独提示覆盖率|
|平均履约耗时|完成订单有效履约耗时合计 / 具有有效耗时的完成订单数|缺失耗时不按 0，单独提示覆盖率|
|履约车辆覆盖率|统计周期内至少完成一单的 Robotaxi 数 / 截止点有效 Robotaxi 数|这是覆盖率，不命名为资产利用率|
|资产时间利用率|Robotaxi 有效履约时间 / Robotaxi 可运营时间|没有状态时间事实时标为不可计算|
|空驶里程占比|非载客运营里程 / 全部运营与履约里程|需要行驶任务类型完整|

旧“供给任务完成率”把准入、投放和运营行驶记录混入同一分母，停止作为经营结论使用。供应过程应分别统计生产、交付、准入和投放的计划达成率。
